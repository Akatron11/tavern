# Tavern App

Taverna için basit bir muhasebe uygulamasi.

## Ne Yapar?

- Gunluk ciro ve gider kaydeder
- Personel listesi
- Veresiye takibi

## Nasil Calistirilir

```
flutter pub get
flutter run
```

Android emulator acik olmali.
